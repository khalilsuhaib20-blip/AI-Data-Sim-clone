Update the existing web application "AI Data Company Simulator Portfolio" with a secure admin interface and a public visitor interface.

The platform simulates my work across multiple companies where I perform roles such as Data Analyst, Data Engineer, Data Scientist, ML Engineer, and Power Platform Architect.

Visitors should only be able to view my work. Only I (admin) can modify data.

---

TECH STACK

Frontend:
React + Tailwind

Backend:
Python FastAPI

Database:
SQLite (MVP)

Authentication:
JWT based authentication

LLM Providers:
OpenAI API
Anthropic API

Environment variables:

OPENAI_API_KEY
ANTHROPIC_API_KEY
JWT_SECRET_KEY

---

AUTHENTICATION SYSTEM

Create a secure admin authentication system.

Only one admin user is required for MVP.

Create a users table:

id
username
password_hash
role
created_at

Role values:
admin
viewer

Password must be hashed using bcrypt.

---

AUTH ENDPOINTS

POST /auth/login
POST /auth/logout
GET /auth/me

Login returns a JWT token.

The token must be stored in the frontend and sent in headers:

Authorization: Bearer <token>

TASK GENERATION USING LLM

Admin can click:

"Generate Task"

Backend endpoint:

POST /generate-task

The backend sends prompt to OpenAI or Anthropic API.

Prompt example:

"You are simulating a real technology company team.

Generate a realistic task request for a data professional.

Company industry: {industry}

Possible roles requesting work:
Stakeholder
Software Engineer
Data Engineer
Data Scientist
ML Engineer
Analyst

Return JSON with:

title
description
requested_by
priority
project_area
recommended_role
deliverables"

Parse the JSON and create a new task.
